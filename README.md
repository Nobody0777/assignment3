# Assignment 3: Kubernetes Orchestration & Full DevOps Pipeline

A 3-tier microservice application (Nginx -> Flask API -> MySQL) deployed using Docker and Kubernetes (Minikube).

## Services
- **Nginx** - Reverse proxy (Port 80)
- **Flask API** - REST API backend (Port 5000)
- **MySQL** - Database (Port 3306)

## Quick Start
```bash
./start.sh
```
